Panduan Penggunaan Sistem

Ringkasan cara memakai Website Pendataan Sarana Prasarana MAN 3 Kota Banda Aceh, mulai dari input data, edit data, upload foto, sampai cetak laporan.

Versi Siap Pakai Gunakan panduan ini untuk petugas/admin yang baru mulai mengelola data Sapras.

1 Alur Kerja Utama

  1. Login menggunakan akun admin.
  2. Input sarana yang masih baik atau input kerusakan baru.
  3. Pantau ringkasan melalui Dashboard.
  4. Kelola data dari halaman Data Masuk.
  5. Update status penanganan jika sudah diproses.
  6. Cetak laporan dari halaman Laporan.

2 Input Sarana Kondisi Baik

  1. Buka menu Input Data.
  2. Isi bagian Input Kondisi Baik.
  3. Pilih lokasi, ruangan, kategori, nama barang, jumlah, dan satuan.
  4. Klik Simpan Kondisi Baik.
  5. Data akan muncul di Dashboard, Data Masuk, dan Laporan.

3 Input Kerusakan

  1. Buka menu Input Data.
  2. Isi data petugas, lokasi, ruangan, kategori, jenis kerusakan, jumlah, prioritas, dan rekomendasi.
  3. Upload maksimal 3 foto awal kerusakan. Foto akan dikompres otomatis agar proses kirim lebih cepat.
  4. Klik Simpan Data.
  5. Status awal otomatis menjadi Belum Diproses.

4 Edit Data Utama

  1. Buka menu Data Masuk.
  2. Cari data berdasarkan ID, ruangan, barang, atau filter.
  3. Klik tombol Edit.
  4. Perbaiki data teks yang salah.
  5. Untuk data kerusakan, foto awal dan foto setelah penanganan juga bisa diganti atau dihapus.
  6. Klik Simpan Perubahan.

5 Edit Status & Foto Bukti

  1. Buka Data Masuk.
  2. Klik Status pada data kerusakan.
  3. Pilih Belum Diproses, Diproses, Selesai, atau Ditunda.
  4. Isi catatan tindak lanjut.
  5. Upload foto setelah penanganan sebagai bukti terbaru. Foto juga dikompres otomatis sebelum dikirim.
  6. Klik Simpan Status.

6 Cetak Laporan

  1. Buka menu Laporan.
  2. Gunakan filter tanggal, lokasi, kondisi, prioritas, atau status.
  3. Klik Cetak / PDF.
  4. Pilih printer atau Save as PDF.
  5. Gunakan layout Landscape agar tabel lebih rapi.
Catatan penting: jangan mengubah ID Kerusakan, ID Kondisi, dan Timestamp secara manual di Spreadsheet. ID dipakai sistem untuk mencari dan memperbarui data yang benar.

Detail Laporan

Informasi lengkap laporan kerusakan.

Edit Status Penanganan

Perbarui status tindak lanjut laporan kerusakan.

Upload bukti foto terbaru
Opsional. Gunakan untuk bukti setelah perbaikan, pemeliharaan, atau penggantian selesai. Maksimal 3 foto.
Jika status dipilih Selesai dan tanggal selesai dikosongkan, sistem akan mengisi tanggal hari ini secara otomatis di Spreadsheet.

Edit Data

Perbarui data utama dan foto jika ada kesalahan input.

ID dan tanggal input tidak bisa diedit agar riwayat data tetap aman. Jika foto diganti, sistem akan menyimpan foto baru dan memperbarui link di Spreadsheet. File lama tidak dihapus otomatis dari Google Drive.
